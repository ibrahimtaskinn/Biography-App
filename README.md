# BiographyApp
My biography application has a basic design foundation and consists of five pages: Home, About, Gallery, Contact, and Blog.

## Project Features

- MVVM
- Glide
- Bottom Navigation
- DataBinding
- Dagger Hilt 
- Coroutines
- WebView
